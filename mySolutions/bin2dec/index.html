Bin2Dec

The binary system is a base-2 numeral system that uses only two digits, 0 and 1, to represent numbers. It is the foundation of all modern computing systems because digital devices operate using two states (on and off), which correspond to these two digits.

How it works

In binary, each position of a digit represents a power of 2, starting from the rightmost digit. For example, the binary number 1011 can be broken down as:

  • 1 x 2^3
  • 0 x 2^2
  • 1 x 2^1
  • 1 x 2^0

Binary to decimal conversion

To convert a binary number into a decimal number, you sum the products of each binary digit (bit) by its corresponding power of 2. Using the example above:
(1 x 2^3)+(0 x 2^2)+(1 x 2^1)+(1 x 2^0) = 8 + 0 + 2 + 1 = 11
Thus, the binary number 1011 equals 11 in decimal. This process can be applied to any binary number to convert it to its decimal equivalent.
